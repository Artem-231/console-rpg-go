# Console RPG & REST API Service on Go

Модульное приложение на Go, объединяющее консольную RPG-игру и RESTful API сервис.
Проект демонстрирует реализацию слоистой архитектуры, интеграцию с PostgreSQL и конкурентную обработку HTTP-запросов.

## Стек технологий

- **Язык:** Go (Golang) 1.22+
- **База данных:** PostgreSQL (драйвер `lib/pq`)
- **Архитектура:** Слоистая (Handler -> Service -> Repository)
- **Безопасность:** Хеширование паролей (`bcrypt`)
- **Конфигурация:** Управление переменными окружения (`godotenv`)
- **Контроль версий:** Git & GitHub Flow

## Ключевой функционал

### Игровое ядро (CLI)
- **Аутентификация:** Система регистрации и авторизации с безопасным хранением учетных данных.
- **Инвентарь:** Управление предметами, слоты экипировки и динамическое изменение характеристик.
- **Экономика:** Механики торговли (покупка/продажа) с валидацией транзакций на уровне БД.
- **События:** Генерация случайных событий (RNG) и интерактивное взаимодействие с миром.

### REST API
Встроенный HTTP-сервер обеспечивает удаленное взаимодействие с игровым состоянием через JSON:
- `GET /status` — Получение текущего статуса и характеристик игрока.
- `POST /create` — Регистрация нового пользователя через API.
- `POST /buy` — Выполнение игровых транзакций (покупка предметов).
- **Middleware:** Логирование входящих запросов и времени выполнения.

## Установка и запуск

1. **Клонирование репозитория:**
   ```bash
   git clone [https://github.com/Artem-231/console-rpg-go.git](https://github.com/Artem-231/console-rpg-go.git)

2. **Настройка окружения:**
   Убедитесь, что PostgreSQL запущен. Создайте файл `.env` в корне проекта:
   ```env
   SERVER_PORT=:8080
   DB_FILE=postgres://user:password@localhost:5432/dbname?sslmode=disable

3. **Запуск приложения:**
   ```bash
   go run main.go
